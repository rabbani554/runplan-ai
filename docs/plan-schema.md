# training_plan.json — Schema Reference

This file is generated by Claude and read by `scripts/upload_plan.py` to upload your plan to COROS.

You can manually edit it before uploading if you want to tweak sessions, swap days, or adjust distances.

---

## Top-level structure

```json
{
  "plan_name": "string",
  "overview": "string",
  "total_weeks": 12,
  "sessions": [ ... ]
}
```

| Field | Type | Description |
|---|---|---|
| `plan_name` | string | Name shown in COROS app |
| `overview` | string | Short description of the plan |
| `total_weeks` | integer | Total length of the plan |
| `sessions` | array | All individual training sessions |

---

## Session object

Every session has a type that determines which fields are required.

```json
{
  "week": 1,
  "day": "TUE",
  "name": "W1 Easy 5km",
  "type": "easy_run",
  "sport": "running",
  "description": "Keep this conversational — if you can't hold a full sentence, slow down.",
  "steps": [ ... ]
}
```

| Field | Type | Description |
|---|---|---|
| `week` | integer | Week number (1 = first week of plan) |
| `day` | string | Day code: `MON` `TUE` `WED` `THU` `FRI` `SAT` `SUN` |
| `name` | string | Session name shown on the watch |
| `type` | string | Session type (see below) |
| `sport` | string | `"running"` or `"strength"` |
| `description` | string | Coaching tip or motivation — shown in COROS app when you tap the workout |
| `steps` | array | Steps for running sessions |
| `exercises` | array | Exercises for strength sessions |

---

## Session types

### Running sessions

| Type | Description |
|---|---|
| `easy_run` | Standard Z2 easy run |
| `long_run` | Weekly long run (Z2, longer distance) |
| `recovery_run` | Very easy Z1 run (after hard sessions) |
| `tempo` | Threshold effort run |
| `intervals` | Structured interval repeats |
| `easy_strides` | Easy run followed by short fast strides |
| `marathon_pace` | Goal marathon pace effort |
| `time_trial` | Race-effort test (e.g. 5km TT) |

### Strength sessions

| Type | Description |
|---|---|
| `strength` | Gym/bodyweight strength session |

---

## Step types (running)

### `warmup`

```json
{ "step_type": "warmup", "duration_s": 300 }
```

| Field | Description |
|---|---|
| `duration_s` | Duration in seconds (300 = 5 min) |

---

### `run`

```json
{ "step_type": "run", "distance_km": 5.0, "hr_low": 140, "hr_high": 158 }
```

| Field | Description |
|---|---|
| `distance_km` | Target distance in kilometres |
| `hr_low` | Lower HR boundary (bpm) |
| `hr_high` | Upper HR boundary (bpm) |

---

### `cooldown`

```json
{ "step_type": "cooldown", "duration_s": 300 }
```

Same as warmup.

---

### `interval_group`

```json
{
  "step_type": "interval_group",
  "reps": 6,
  "distance_m": 800,
  "hr_low": 185,
  "hr_high": 195,
  "rest_s": 90
}
```

| Field | Description |
|---|---|
| `reps` | Number of repetitions |
| `distance_m` | Distance per rep in metres |
| `hr_low` | Lower HR boundary for effort (bpm) |
| `hr_high` | Upper HR boundary for effort (bpm) |
| `rest_s` | Recovery time between reps in seconds |

---

### `strides`

```json
{ "step_type": "strides", "reps": 6, "distance_m": 100, "rest_s": 60 }
```

Short fast accelerations. HR is open (no target). Always 100m per stride.

| Field | Description |
|---|---|
| `reps` | Number of strides |
| `distance_m` | Distance per stride (usually 100) |
| `rest_s` | Walk/jog recovery between strides |

## HR zone reference

Zones are calculated from your actual COROS data. These are approximate ranges for reference:

| Zone | Purpose | % HRmax |
|---|---|---|
| Z1 Recovery | Very easy, post-race | < 68% |
| Z2 Easy | Base building, long runs | 68–78% |
| Z2 Long | Long run top end | 68–80% |
| Z4 Tempo | Threshold, comfortably hard | 85–92% |
| Z5 Intervals | Hard efforts, VO2max | 93–100% |
| Marathon Pace | Goal race effort | 78–85% |

## Strength session structure

```json
{
  "week": 4,
  "day": "MON",
  "name": "W4 ST-A Lower Body",
  "type": "strength",
  "sport": "strength",
  "exercises": [
    { "key": "squat",         "sets": 3, "value": 10, "rest_s": 45 },
    { "key": "walking_lunge", "sets": 3, "value": 10, "rest_s": 45 },
    { "key": "plank",         "sets": 3, "value": 30, "rest_s": 30 }
  ]
}
```

| Field | Description |
|---|---|
| `key` | Exercise identifier (see list below) |
| `sets` | Number of sets |
| `value` | Reps (for rep-based) or seconds (for hold-based) |
| `rest_s` | Rest between sets in seconds |
